# **App Name**: Contract Clarity

## Core Features:

- Contract Explanation: Breaks down Sui Move smart contracts function by function, converting them into UML sequence diagrams for easier understanding. Includes a tool powered by genai that determines whether a concept requires explanation.
- UML Diagram Display: Displays generated UML sequence diagrams for each smart contract function.
- Diagram Download: Allows users to download the UML sequence diagrams as images.
- Contract Link Input: Provides a field for users to paste Sui explorer smart contract links.
- Community Blog: A dedicated blog page where users can discuss smart contracts, referencing explanations generated within the web app.
- Light/Dark Mode Switch: Allows users to switch between light and dark UI modes.
- Optional Login: Allows users to log in

## Style Guidelines:

- Primary color: Saturated blue (#4285F4) to convey trustworthiness and intelligence.
- Background color: Light gray (#F5F5F5) for light mode and dark gray (#333333) for dark mode.
- Accent color: Analogous light blue (#03A9F4) to highlight interactive elements.
- Headline font: 'Space Grotesk' sans-serif font to convey a modern look and feel. Body font: 'Inter' sans-serif for body text. 
- Use simple, geometric icons to represent contract elements and actions.
- Clean and structured layout with clear sections for input, explanation, and diagram display.
- Subtle transitions and animations to enhance user experience.
